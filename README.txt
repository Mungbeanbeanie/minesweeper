A classic Minesweeper game built with vanilla HTML, CSS, and JavaScript. This project is a simple, self-contained, and customizable implementation of the timeless puzzle game.

üöÄ Features
Classic Gameplay: Left-click to reveal squares and right-click to plant or remove flags.

Customizable Settings: Easily change the board size (NxN) and the number of mines using the input fields.

Flood-Fill: Automatically reveals all adjacent empty squares when you click an empty one.

Mine Counter: Tracks how many mines are left to find (based on your flag count).

Win/Loss Detection: Get a clear message whether you win or lose.

No Dependencies: Runs entirely in the browser with no external libraries needed.

You can run this project in two simple ways:

1. Run Locally
Clone this repository or download the three files (index.html, style.css, game.js) into the same folder.

Open the index.html file in your favorite web browser.

2. Deploy with GitHub Pages (Recommended)
Create a new repository on GitHub and push these three files to it.

In your repository's settings, go to the Pages tab (in the left sidebar).

Under "Build and deployment", select the Source as Deploy from a branch.

Select your main (or master) branch and the /root folder, then click Save.

Wait a minute or two, and your game will be live at the URL shown on that page!
